The project I have carried out have all been throught university. The first project was an arduino group project which we have to create anything we wanted aslong as it was functionable and contain as couple of compulsory elements (i.e has to contain buttons, LEDs, sensors etc.)

My group decided to create a jukebox. It would play a song when one of the five buttons was pressed. We all coded our own songs and then, as a group, worked together to wire the arduino and connect it to processing.

I really enjoyed this project as it allows me to add my interest of music to computing as we coded songs. I also used my creative knowledge to help design what the jukebox would look like and how it was going to work all together.
